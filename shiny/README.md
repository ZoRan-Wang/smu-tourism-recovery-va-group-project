# Tourism Classification Explorer

This folder is self-contained.

## Run

```r
shiny::runApp()
```

Or point R directly at this folder:

```r
shiny::runApp("path/to/this/shiny")
```

## Required structure

- `app.R`
- `data/tourism_decision_tree_ready.csv`
- `outputs/`

Keep these files together after uploading to your repository.
